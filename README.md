# Khamsa 🏺
A premium, modern e-commerce marketplace dedicated to authentic craftsmanship, local pottery, and traditional decor. 

## ✨ Key Features
* **Role-Based Access Control (RBAC):** Secure Admin dashboard for full CRUD operations on marketplace inventory.
* **Persistent Cart State:** Dynamic shopping cart that retains user selections across sessions using LocalStorage.
* **Modern UI/UX:** Apple-inspired design featuring glassmorphism navigation, premium typography (Poppins), and interactive CSS animations.
* **RESTful Integration:** Seamless asynchronous data fetching using Fetch API with MockAPI.

## 🚀 Tech Stack
* **Frontend:** React.js, React Router DOM
* **State Management:** React Context API
* **Styling:** Custom CSS, CSS Variables, Keyframe Animations
* **Backend:** MockAPI (REST endpoints)

## 💻 Getting Started
1. Clone the repository
2. Run `npm install` to download dependencies
3. Run `npm install canvas-confetti` for the checkout animation
4. Run `npm run dev` to start the local development server
